# FakeMessenger

**FakeMessenger** — консольное приложение на C#, имитирующее работу мессенджера.  
Позволяет создавать контакты и групповые чаты и симулировать обмен сообщениями.

## Возможности

- Управление контактами: создание новых контактов с именем и фамилией.
- Создание групп: объединение контактов в группы.
- Личные чаты: создание личного чата с контактом.
- Обмен сообщениями: отправка текстовых и симуляция мультимедийных сообщений.
- История сообщений: просмотр переписки в чате.
- Сохранение и загрузка: данные (контакты, чаты, сообщения) сохраняются в XML-файл `Save.Messager.xml` и загружаются из него.
- Интерактивный поиск: удобный выбор действия с поиском и обновлением результатов прямо в консоли.

## Технологии

- Платформа: `.NET 10`
- Язык: `C#`
- Сохранения: `XML-файл` через `XDocument`

## Установка и запуск
> [!WARNING]
> Рекомендуется использовать только официальные релизы. Сборки из GitHub Actions могут быть нестабильными.
1. **Перейдите в [последний релиз](https://github.com/AlmasikkMe/FakeMessenger/releases/latest)** или в последний workflow в [GitHub Actions](https://github.com/AlmasikkMe/FakeMessenger/actions/workflows/dotnet-desktop.yml)
2. **Скачайте файл запуска** для вашей OS
3. **Поместите** скачанный файл **в любую папку**
4. **Разрешите запуск:**
   - **Для Windows:** `ПКМ по файлу -> Свойства -> Разблокировать -> Применить`
   - **Для Linux:**
      ```bash
      chmod +x ConsoleFakeChat
      ```
5. **Запустите файл**

## Использование

После запуска открывается главное меню:

- **Создать контакт** — добавление нового пользователя.
- **Создать группу** — объединение нескольких существующих контактов в групповой чат.
- **Перейти в чат** — открытие меню чата с выбраным существующим чатом.
- **Сохранить** — сохранение текущего состояния в XML-файл.
- **Загрузить** — загрузка состояния из XML-файла.
- **Создать чат с контактом** — создание личного чата с выбранным существующим контактом.

Внутри чата доступны команды:
- **Посмотреть историю чата** — вывод всех сообщений.
- **Отправить сообщение** — добавление текстового сообщения.
- **Отправить мультимедиа** — отправка сообщения с указанием типа (например, "photo", "video").

Для выбора пункта меню используются стрелочки на клавиатуре и интерактивный поиск: начните вводить название пункта, и список будет фильтроваться.

## Структура проекта

- `Program.cs` — точка входа.
- `ConsoleUI.cs` — пользовательский интерфейс (меню, диалоги, поиск).
- `Messenger.cs` — основная логика (контакты, чаты, сохранение/загрузка).
- `User.cs` — модель пользователя.
- `Chat.cs` — модель чата (группового или личного).
- `Message.cs` — модель сообщения.
- `ConsoleFakeChat.csproj` — файл проекта для .NET 10.
